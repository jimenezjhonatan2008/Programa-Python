## Descripción de la solución

Este programa en Python analiza una matriz de sesiones de clientes. Cada sesión contiene el ID del cliente, la duración en segundos y la cantidad de clics realizados. Con estos datos, el programa clasifica el nivel de compromiso de cada sesión como **Alto**, **Medio** o **Bajo**.

Criterios usados:

- **Alto:** duración mayor a 180 segundos y más de 8 clics.
- **Bajo:** duración menor a 60 segundos o menos de 3 clics.
- **Medio:** sesiones que no cumplen las condiciones anteriores.

El programa también permite agregar nuevas sesiones manualmente y genera un resumen general con el total de sesiones por nivel de compromiso.
